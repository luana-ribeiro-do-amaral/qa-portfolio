# Test Case – Verify International Footer Link Redirect

## Test Case ID

TC-FOOTER-001

## Module

Footer Navigation

## Priority

Medium

## Preconditions

User is on the Brazilian Domino's Pizza homepage (dominos.com.br).

## Test Steps

| Step | Action                         |
| ---- | ------------------------------ |
| 1    | Open a browser                 |
| 2    | Navigate to dominos.com.br     |
| 3    | Scroll to the footer section   |
| 4    | Click the "Internacional" link |

## Expected Result

The user should be redirected to the Domino's international page displaying available countries, regions, or global markets.

## Actual Result

The user is redirected to a blank page.

## Status

Failed
