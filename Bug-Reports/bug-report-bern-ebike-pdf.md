# Bug Report – PDF download for the E-Bike Map is not available

## Bug ID
TC-001

## Summary
PDF download for the E-Bike Map is not available

## Severity
Critical

## Environment
- OS: Windows 11
- Browser: Google Chrome 136
- Website: The official page of Bern 

## Preconditions
User is on the Practical Information page.

## Steps to Reproduce
1. Open the Home page.
2. Locate the Information section.
3. Select the Practical Information section.
4. Scroll to the E-Bike Map.
5. Click Download PDF now.

## Expected Result
The PDF is downloaded correctly.
## Actual Result
An error message is displayed instead.

## Frequency
5/5 times (100%)
